# Node Swapit

Easily swap files

# Installation
```
$ npm i --save swapit
```

# Usage
```
$ swapit a.txt b.txt
```

Is the same as:

```
$ mv a.txt a_.txt
$ mv b.txt a.txt
$ mv a_.txt b.txt
```
